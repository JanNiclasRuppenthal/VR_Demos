# Reverse Angry Birds Game
**Developers:**
- Michael Feldmann 
- Philipp Geier
- Jan Niclas Ruppenthal

## Exercise Description

In the second exercise of the module "VR and 3D Interactions", we were tasked with implementing various 3D interactions across several task parts. All parts of this exercise were divided into different scenes.
In the final part of this exercise, the player had to instruct a bird on which tree it should circle. This is done by pointing at the desired tree with the controller. We wanted to add more fun to this idea, which is why we developed "Reverse Angry Birds". The red bird is launched into the field using a slingshot, and you can then shoot it down with your bow. The green pigs from Angry Birds were attached to the arrowheads.


# Controls

## Unity Editor Controls:
  - **Spacebar (Task part g)**: Change perspective
  - **D Key (Task part h)**: Bird dies


## VR Controls:
  - **Grab Button**: Draw the bow or release the string
  - **Right Stick** - **Forward**: Teleportation
      - **Other directions**: Rotation
  - **Left Stick**: Movement
  - **Cancel Teleportation**: Press the grab button during teleportation
  - **Select tree**: Point at the tree with the left controller and then press the trigger button
