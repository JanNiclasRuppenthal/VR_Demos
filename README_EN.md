# Ames Room
**Developers:**
- Michael Feldmann 
- Philipp Geier
- Jan Niclas Ruppenthal

## Exercise Description
In the first exercise of the module "VR and 3D Interactions", we were tasked with implementing an [Ames room](https://en.wikipedia.org/wiki/Ames_room) both in the editor and in VR. We decided on a space theme.

# Controls:

## Editor:
  - **W**: Toggle a partition wall in the Ames room
  - **P**: Start or pause the astronauts' movement
  - **Spacebar**: Start or pause the camera movement
  - **R**: Reset the camera position

## VR:
  - **Left Joystick**: Movement 
  - **Right Joystick**: Rotation by 45°

  - **Right Controller**:
      - **A**: Toggle the front wall
  - **Left Controller**:
      - **X**: Start or pause the astronauts' movement
      - **Y**: Toggle a partition wall in the Ames room
